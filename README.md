# Hệ thống quản lý an toàn thông tin sinh viên

## Cách chạy project

1. Cài Python
2. Cài MySQL
3. Cập nhật thông tin MySQL trong `config.py`
4. Chạy file `setup_database.py` để tạo database và bảng
5. Cài thư viện:
   pip install mysql-connector-python
6. Chạy chương trình:
   python main.py
7. Cài thư viện Customtkinter:
   pip install customtkinter
8. Chạy giao diện:
   python gui.py
